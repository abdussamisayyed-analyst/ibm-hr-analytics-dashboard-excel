# 📊 IBM HR Analytics Dashboard (Excel)

## Project Overview
This project is an interactive HR Analytics Dashboard created in Excel to analyze employee attrition, workforce demographics, overtime trends, and salary insights.

## Features
- Employee Attrition Analysis
- Salary & Income Insights
- Overtime Percentage Tracking
- Department & Job Role Analysis
- Interactive Charts and KPI Cards

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- KPI Cards
- Conditional Formatting
- Slicers
